# README #

This repository contains a bower module.
It allows previewing the uploaded image in a form.

***

### How to use ###

    Create an input with the name 'picture' and class 'itm-input-file'.
    Then create a label for this input with a 'btn' class from bootstrap.
    Then have a HTML img element with the class 'itm-view-picture'.
